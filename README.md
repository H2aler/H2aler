# 안녕하세요! 👋

저는 한국에서 중학교 졸업 후 **프랑스**에서 고등학교와 대학교를 졸업한 김재현입니다.

## 학력
- **고등학교**: 이과 특별전공 엔지니어링(전기, 물리, Solid Works)
- **대학교**: 컴퓨터 공학 전공

## 경력
### 세아상역 인턴쉽
- **기간**: 6개월
- **업무**: Microsoft Power Automate를 활용한 RPA(로봇 프로세스 자동화) 기반 시스템 설계
- **경험**: 애자일 방법론 이해, 스프린트 회의 및 팀원들과의 협업 경험
- **추천서**: 보유

### 웹 개발
이후 한국에 재정착하여 웹 개발을 취미로 시작했습니다. 작은 병원에서 반응형 웹사이트 제작 의뢰를 받아 일하며 이 분야가 제 적성에 맞다는 것을 알게 되었습니다. 현재는 **한불 언어마을**에서 번역 업무와 웹 개발, 유지보수를 맡고 있습니다. 

웹 개발을 시작한 지 얼마 되지 않아 아직 서툰 부분이 있지만, 처음에는 어려웠던 작업들이 지금은 다양한 프레임워크를 배우고 사용하는 것이 재미있어졌습니다.

## 다짐
이러한 경험을 바탕으로 **귀사(산업기능요원 지원 회사)**에서 **프론트엔드 개발자**로서 일할 기회를 주신다면 최선을 다할 것을 약속드립니다.

감사합니다! ☺️
